We use this command to start Apache Jmeter.

You can use $VAR style (even on Windows) for references to Jenkins-defined environment variables.

1) Start tests with load generator with docker on Jenkins slave:
docker run --rm -v "$WORKSPACE":/data:rw -w "/data/$PERFCI_WORKING_DIR" docker-perf.eng.pek2.redhat.com:5000/perfci-agent jmeter

2) Start tests with load generators in OpenShift
ssh $WORKSPACE/openshift/run_test.sh $WORKSPACE $BUILD_NUMBER
